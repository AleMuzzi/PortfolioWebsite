# Sviluppatore / Progettista R&D presso Spark Security

## Summary
Sviluppatore R\&D orientato alle performance, specializzato nella stabilità di sistema a basso livello e in integrazioni di sicurezza ad alta precisione. Esperto nello sviluppo di bridge tra architetture C++ e C\# per garantire la sincronizzazione dei dati in tempo reale con precisione al singolo frame in contesti di videosorveglianza critici.

## Azienda
Spark Security Srl - https://www.spark-security.com/

## Periodo
Maggio 2018 — Ottobre 2019

## Cosa ho fatto

### **Stabilità di Sistema e Precisione**

* **Progettazione di un motore di sincronizzazione ad alta precisione** per allineare gli allarmi delle telecamere ai relativi frame video all'interno della suite di gestione. Questo ha garantito un recupero degli eventi accurato al singolo frame, migliorando sensibilmente l'affidabilità degli audit di sicurezza e delle indagini sugli incidenti.
* **Risoluzione di un complesso memory leak latente** nello strato di interoperabilità (interop) C++/C\#, causato da una gestione errata dei buffer dei frame ad alta frequenza. Questa analisi tecnica approfondita ha eliminato i crash periodici del sistema, garantendo stabilità a lungo termine per le installazioni enterprise.
* **Ottimizzazione dell'elaborazione video in tempo reale** attraverso il refactoring e il performance tuning dell'interfaccia .NET/XAML, riducendo il carico sulla CPU e rendendo l'esperienza utente più fluida durante la riproduzione multi-stream.

### **Architettura e Sviluppo SDK**

* **Sviluppo di un SDK di comunicazione affidabile** per standardizzare le interazioni con il database backend, creando un componente modulare che ha accelerato lo sviluppo interno dei futuri moduli della piattaforma.
* **Ampliamento della compatibilità hardware** integrando i controlli avanzati per telecamere PTZ (Pan-Tilt-Zoom) nella piattaforma principale, offrendo agli utenti finali una gestione dell'hardware fluida e a bassa latenza.
* **Guida delle iniziative di code review** basate sul design pattern MVVM per garantire elevati standard di conformità e mantenere il codice manutenibile all'interno del dipartimento R\&D.
