# Tirocinio per tesi magistrale presso Maps Group
## Summary
Sviluppo di un motore di ricerca semantico per i report di incidenti in ambito sanitario.
## Period
Settembre 2021 — Febbraio 2022
## What i did
- Studio e sviluppo di un motore di ricerca semantico per cercare report di incidenti nel settore sanitario.
- Il motore di ricerca fa parte di un progetto più ampio per lo sviluppo di una piattaforma di gestione intelligente per i report di incidenti commissionata dalla regione Emilia Romagna.
